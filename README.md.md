This is a console-based Tic Tac Toe game developed using Java.  
It allows two players (X and O) to play by entering row and column values.

---

## Features
- 2 player mode (X vs O)
- Input validation
- Winner detection
- Draw detection
- Play again option

---

- Java  
- Eclipse / VS Code  
- Git & GitHub  

---

## How to Run

### Using Eclipse / VS Code
1. Open the project folder  
2. Run `TicTacToe.java`

### Using Terminal
# TicTacToe-Java
Java Tic-Tac-Toe Game


---

## How to Play
- Enter row and column numbers between **0 and 2**  
- Player **X** goes first, then **O**  
- The game announces the winner or a draw  

Example input:
0 1
